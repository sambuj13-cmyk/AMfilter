</p> # AMfilter

AMFilter is a modern, front-end YouTube search tool built with HTML, CSS, and JavaScript. It supports dark/light mode, infinite scroll, Shorts/Long-video filtering, PG-safe search, and a smooth neon-styled UI powered by the YouTube Data API v3. Lightweight, fast, and fully responsive.
